
## Button 按钮

### 使用指南

``` js
import { Button } from 'od-ui';
Vue.use(Button);
```
### 代码演示
#### 按钮类型

支持`default`、`primary`、`info`、`warning`、`danger`五种类型，默认为`default`


``` html
  <button>默认尺寸</button>
  <button size="medium">中等按钮</button>
  <button size="small">小型按钮</button>
```

## 内置过渡动画

Element 内应用在部分组件的过渡动画，你也可以直接使用。在使用之前请阅读 <a href="https://cn.vuejs.org/v2/api/#transition" target="_black">transition 组件文档</a>
### 使用方法
``` html
<transition name="bounce">
  ...
</transition>
```
### Attributes
name | intro
---|---
bounce | 弹跳
flash | 闪烁
pulse | 跳动
rubberBand | 弹力
shake | 震动
swing | 摇摆
tada | 快速震动
wobble | 摇晃
jello | jello
heartBeat | 心跳
flip | 翻转跳跃

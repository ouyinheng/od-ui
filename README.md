<h3 align="center" style="margin: 30px 0 35px;">Mobile UI Components built on Vue</h3>

### Install

```
npm i od-ui -S
or
yarn add od-ui -S
```

### Quickstart

```
import Vue from 'vue';
import OdUI from 'od-ui';
import 'od-ui/lib/od-ui.css';

Vue.use(OdUI);
```
